#Turn all the lights on game 
this project is a puzzle game written in C.

##Objective
the goal of the game is to turn all the lights on in the grid.

##How It Works 
-The game uses a grid system.
-Each move toggles a light(neighbour lights)
-The player wins when all lights are turned on.

##How To Compile

using GCC:
gcc GRAPH_SEARCH.c HashTable.c SpecificToProblem.c Standart_Search.c -o lights_out_solver
./lights_out_solver
